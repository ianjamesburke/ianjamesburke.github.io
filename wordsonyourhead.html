Words On Your Head

Custom embroidered hat store. Browse 24 curated designs or type anything you want — a live preview renders instantly, you check out with Stripe, and Printful handles the rest.

wordsonyourhead.com
Astro React Stripe Printful API Railway TailwindCSS v4
Custom Hat Builder
Type anything (up to 40 characters across 3 lines). A live hat preview renders immediately in the browser — corduroy texture, eyelets, crown, and all. Color swaps update both the preview and the Stripe checkout metadata in one go.
Payments & Fulfillment
Stripe Checkout handles payment and shipping collection. On success, a webhook fires to the Printful API to create a draft order with the embroidery PNG attached. Custom Stripe webhook verification fallback works around an Astro Node adapter raw-body limitation.
Embroidery Pipeline
Custom text is converted server-side from SVG to PNG using @resvg/resvg-js with a bundled Liberation Sans Bold font — no browser font loading. Dynamic sizing adjusts character scale by text length. The resulting PNG is sent to Printful as the embroidery file.
