About

I specialize in rapid-iteration full-stack web applications. I've touched every part of the software development lifecycle — scoping projects with non-technical clients, clearly communicating trade-offs and timelines, shipping and iterating MVPs, and establishing CI/CD pipelines for long-term support.

I gravitate toward Rust for systems work, Svelte and React for interfaces, and Python for anything that needs to move fast. I build AI-powered tools with OpenAI, Claude, and Gemini APIs — prompt engineering, RAG pipelines, and intelligent automation are part of the daily toolkit.

I started in social media content production — directing, editing, and producing 1000+ video ads for 7-figure DTC campaigns. That background means I think about software the way a producer thinks about a shoot: what's the deliverable, when does it need to ship, and what's the simplest path to something that works.

Currently building Plexi, shipping Kill Tony Archive, and figuring out what the terminal looks like when AI is a first-class citizen.
